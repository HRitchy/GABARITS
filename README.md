# Quiz fonction de contrôle (Next.js)

Application Next.js ludique pour apprendre la fonction de contrôle de chaque image d'attache.
9 questions, chaque bonne réponse rapporte 2,2225 points, note finale sur 20.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ensuite ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.

## Build de production

```bash
npm run build
npm start
```
